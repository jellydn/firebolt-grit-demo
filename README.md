## [Grit](https://grit.io)

### List all patterns

```sh
grit list
```

### Check the current pattern

```sh
grit check
```

### Describe the pattern

```sh
grit patterns describe <pattern>
```

### Apply the pattern

```sh
grit apply <pattern>
```

### Edit the pattern with Grit Studio

```sh
grit pattern edit
```
